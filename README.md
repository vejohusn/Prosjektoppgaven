# Prosjektoppgaven
Prosjektoppgave - Python ved USN

Jeg har valgt å løse prosjektppgaven ved å legge inn alle deloppgavene som 
funksjoner i en meny-fil som gjør at man kan laste alle funksjonalitetene fra
en meny. Litt mer slik jeg tenker det ville vært hvis man skal lage et program
for en bedrift. 

Jeg har brukt funksjonen: "print("\033[2J\033[H")" for å rengjøre skjermbildet i 
terminalen mellom hver gang menyen skrives. Dette for at det skal se "renere" ut. 
Jeg vet ikke om denne funksjonaliteten fungerer på alle operativsystem, men jeg har
testet den på terminal i ubuntu 24.12 og i Windows 11 med anaconda.
